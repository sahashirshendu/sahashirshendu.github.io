Some useful linux commands.

Updating the system:#

Debian, Ubuntu & their derivatives:*#

1

sudo apt update && sudo apt upgrade

*For KDE Neon,

1

sudo pkcon refresh && sudo pkcon update

Fedora:#

1

sudo dnf update

Arch Linux & distributions based on it:#

1

sudo pacman -Syu

or, if yay is installed,

1

yay

Installing a package:#

Debian, Ubuntu based distros:#

1

sudo apt install packagename

Fedora:#

1

sudo dnf install packagename

Arch based distros:#

1

sudo pacman -S packagename

For example, for installing gfortran,

1
2
3

sudo apt install gfortran # Ubuntu
sudo dnf install gfortran # Fedora
sudo pacman -S gcc-fortran # Arch

Summary - Setting up Fortran in Ubuntu:#

In terminal, run:

1
2
3

sudo apt update && sudo apt upgrade
sudo apt clean
sudo apt install gedit gfortran gnuplot

Next create a folder where the files are to be stored and go there,

1
2

mkdir ~/Fortran
cd ~/Fortran

Create the fortran file,

1

gedit file.f # gedit or nano or micro or whatever editor you like

Compile and run the file,

1
2
3
4
5
6
7

# Compile #
f77 file.f # for Fortran 77 Files
f95 file.f90 # for >= Fortran 90 Files
gfortran file.f # generic

# Run #
./a.out

One can even set up a bash function to compile and execute any fortran file in one shot. To do that insert the following in ~/.bashrc :

ft ()
{
  if [ -x "$(command -v f95)" ]; then
    if [ -x "$(command -v f77)" ]; then
      if [ -f $1 ] ; then
        case $1 in
          *.f90 | *.f95) f95 $1 && ./a.out && rm a.out ;;
          *.f) f77 $1 && ./a.out && rm a.out ;;
        esac
      fi
    else
      if [ -f $1 ] ; then
        case $1 in
          *.f90 | *.f95 | *.f) gfortran $1 && ./a.out && rm a.out ;;
        esac
      fi
    fi
  else
    echo 'Fortran is not installed!'
  fi
}

ft file.f
